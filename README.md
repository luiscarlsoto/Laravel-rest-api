
# About this project


## To get all tasks
````
GET  .../api/showAll
````

## To create a new task
````
POST  .../api/create
````

## To get a specific task
````
GET  .../api/show/{id}
````
## To update a specific task
````
PUT  .../api/showAll/{id}
````
## To delete a specific task
````
DELETE  ../api/delete/{id}
````
